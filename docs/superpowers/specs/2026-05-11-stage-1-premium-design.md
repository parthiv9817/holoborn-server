## What we're building

User triggers capture → over ~3.0s a cyan sacred-geometry sigil materializes on the real floor 1.5m forward, the room reacts to incoming energy via a pre-cast cyan light at the mannequin position, an ambient sub-drone fills the space, then a translucent anatomical mannequin **dissolves into existence bottom-up from the sigil plane** with a cyan emissive edge band moving up the mesh. A chime lands at the moment the dissolve crosses chest height. The mannequin then breathes and slowly Y-rotates; the sigil slowly Y-rotates beneath it. This holds for the full backend processing window (~5–9 min through TRELLIS + Meshy Retexture).

Stages 2 (scan-line reveal) and 3 (awakening) are out of scope today.

---

## The arrival pattern: layered timescales

Per industry research (Doctor Strange portals, Westworld host construction, Blade Runner 2049 Joi, Apple Vision Pro Personas, Half-Life: Alyx), premium VR/MR entity introduction is **layered timescales** — multiple channels each announce the arrival before the entity is fully opaque. Single-channel opacity fade structurally reads as a pop-in regardless of polish.

Four layers, sequenced:

1. **Audio leads** — low sub-drone starts BEFORE any visual change. Highest-leverage immersion lever per Meta Horizon spatial-audio docs.
2. **Anchor first** — sigil completes before mannequin starts to render. Eye accepts the entity because the anchor explains where it came from.
3. **Pre-cast light** — cyan point light ramps up at the mannequin position BEFORE the mesh appears. Room reacts to the arrival before the arrival.
4. **Dissolve from sigil upward** — mannequin uses Shader Graph noise-mask threshold + vertex-Y mask + emissive cyan edge band. Materializes bottom-up FROM the sigil, not in mid-air.

## Locked decisions

### Sigil disc
- **Pattern:** sacred-geometry summoning circle (Kuchiyose / Doctor Strange code) — outer circle, second circle, hexagram, inscribed up + down triangles, inner circle, center dot, vertex dots.
- **Source:** pre-generated PNG at `drafts/summoning_sigil.png` (1024×1024, transparent, cyan #00E0FF). Regen script at `tools/gen_summoning_sigil.py`.
- **Render:** horizontal Quad mesh, **URP Unlit shader, Surface Type: Transparent**, Base Map = sigil PNG, emission color cyan #00E0FF.
- **Behavior:** slow Y-rotation, 1 turn per 30s (12°/sec) via 1-line `transform.Rotate`. Static texture, no procedural animation.
- **Size:** 1.2m diameter (Quad scale 1.2).
- **Position:** ritual anchor X/Z, Y = 0 (real floor height).

### Mannequin
- **Mesh:** existing `silhouette_placeholder.fbx` (Y-Bot, Mixamo-rigged). Eyes closed.
- **Material:** **URP Shader Graph (Unlit target)** combining rim glow + dissolve. ~10 nodes, all drag-connect built-ins, no formulas typed.
  - Rim branch: Fresnel Effect → Multiply (× RimColor × RimIntensity) → Emission
  - Dissolve branch: Position node (object Y) → Compare with DissolveThreshold property → Alpha Clip; second compare within ±0.05 of threshold → emissive cyan for the edge band
  - BaseColor + BaseAlpha → Base Color output
  - Surface = Transparent, Blend = Alpha
  - Exposed properties: BaseColor, RimColor, RimIntensity (Float), BaseAlpha (Float), DissolveThreshold (Float, 0–1)
  - Values at rest: BaseColor=#E8E8EA, RimColor=#00E0FF, RimIntensity=1.2, BaseAlpha=0.4, DissolveThreshold=0.0 (fully visible)
- **Behavior at rest:**
  - Float offset: `position.y = anchor.y + 0.03f`
  - Slow Y-rotation: 1.5°/sec via 1-line `transform.Rotate`
  - Breath: looped Animator clip, ±3% spine scale, 0.5 Hz (2s loop)
- **Position:** 1.5m forward of capture trigger, eye-line height (anchor.y ≈ 1.6m).

### Pre-cast Light
- Unity Light component (Point Light), parented at mannequin position.
- Color cyan #00E0FF, range 2m, intensity animated by coroutine.
- Activates BEFORE mannequin renders to give "room reacts before entity arrives" beat.

### Audio
- **Drone stem:** `drafts/audio/summoning_drone.wav` — 8s seamless loop, low sub-drone (60Hz fundamental + harmonics + slow LFO breath). Plays from t=0 of arrival through Stage 1 hold.
- **Chime stem:** `drafts/audio/summoning_chime.wav` — 1.5s single-shot magical chime (E major triad + shimmer + exponential decay). Plays at t=1.8s when dissolve crosses chest height.
- Both generated by `tools/gen_summoning_audio.py` (first-pass synth; can be swapped for freesound CC0 textured stems later if too clinical).
- Spatialized via Unity AudioSource at sigil position.

### Arrival sequence (~3.0s total)

| t (s) | Event | Tech |
|---|---|---|
| 0.0 | `OnShellManifestEnter` fires from `SpawnRitualController` | event |
| 0.0 | Audio drone starts, AudioSource at sigil position (looped) | AudioSource.Play |
| 0.0 → 0.6 | Spawn sigil Quad at anchor, scale 0→1 + alpha 0→1, ease-out cubic | coroutine |
| 0.4 → 1.4 | Pre-cast cyan Point Light at mannequin position ramps intensity 0 → peak | coroutine |
| 0.6 | Spawn mannequin at anchor, DissolveThreshold = 1.0 (fully clipped), invisible | instantiate |
| 1.0 → 2.8 | DissolveThreshold animates 1.0 → 0.0 ease-out cubic — mesh resolves bottom-up with cyan emissive edge band | coroutine driving material property |
| 1.0 → 2.8 | Mannequin transform.y rises 5cm from `anchor.y - 0.02` to `anchor.y + 0.03` | coroutine (parallel) |
| 1.8 | Audio chime plays at moment dissolve crosses chest height | AudioSource.PlayOneShot |
| 2.8 → 3.2 | Pre-cast light fades to ambient rim level (stays on at low intensity through hold) | coroutine |
| 3.0 | Breath Animator + Y-rotation coroutines take over for Stage 1 hold | Animator + coroutine |

No particles. No VFX Graph. All shader work is built-in node graph (drag-connect).

## Acceptance criteria

On Quest device, with `_DebugStatusSimulator` firing `ShellManifest`:

1. Audio drone begins at t=0 (audible immediately).
2. Cyan summoning sigil fades + scales in on the real floor at anchor over ~0.6s.
3. Pre-cast cyan light appears at mannequin position ~0.4s after sigil (room reacts before entity).
4. At ~1.0s, mannequin begins dissolving into existence bottom-up from the sigil plane with a visible cyan emissive edge band traveling up the mesh.
5. Chime lands at ~1.8s when edge band crosses chest height.
6. By ~2.8s, mannequin is fully visible with **subtle cyan edge rim along silhouette** (NOT uniformly tinted — no Tron-cyan body repeat of 2026-05-11 mistake).
7. Mannequin breathes and slowly Y-rotates.
8. Sigil slowly Y-rotates beneath it.
9. Drone continues quietly through the 5-min hold.
10. No frame drops, no z-fighting, no visible artifacts.

## Risk register

| Risk | Likelihood | Impact | Mitigation |
|---|---|---|---|
| Fresnel + dissolve combo on Quest XR performs poorly | Low | High | Verify on device after shader build; fall back to simpler rim-only + opacity fade if needed |
| Synth audio reads too clinical, breaks immersion | Medium | Medium | Listen in headphones before Quest build; swap to freesound CC0 textured stems if needed (~15 min) |
| Pre-cast Point Light doesn't read through Quest passthrough | Medium | Medium | If invisible, increase intensity or add an unlit emissive cyan halo quad as visible companion |
| Sigil PNG line weight reads wrong on Quest passthrough | Low | Low | `tools/gen_summoning_sigil.py` constants tunable; regen in seconds |
| Y-Bot has no breath clip out of the box | Medium | Medium | Build 2-keyframe spine-scale clip (±3%, 2s loop) |
| Dissolve edge-band intensity too bright or too dim | Medium | Low | Exposed property `DissolveEdgeIntensity` — tune in Inspector |
